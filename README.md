# Description
Flunch is an open source non-alcohol drink for everyone who’s feeling thirsty.

# Recipe
* Water: 0,35.
* Blue Curaçao: 0,14.
* Pineapple juice: 0,28.
* Grapefruit juice: 0,16.
* Lemon juice: 0,05.
* Lemon syrup: 0,02.

Optionally, you can add mint and few slices of lemon and orange into the mix.
